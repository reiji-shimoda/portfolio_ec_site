# ポートフォリオ用ECサイト

ポートフォリオ用にECサイトを作成しました。

## デモ

## 使い方

ユーザーを登録し、ログインする。

購入者側

1. 買いたい商品の個数を選択し、カートに入れるボタンをクリックする。
2. カートボタンまたは購入手続きボタンをクリックする。
3. カート内の商品を確認する画面にて購入ができる。
4. 購入履歴で購入した商品の履歴を見ることができる。

出品者側

1. \register\adminで出品者を登録する。 
2. \login\adminにて出品者ログインする。
3. 商品登録画面で出品する商品の情報を入力する。
4. 登録商品一覧で登録した商品の一覧を見ることができる。
5. 購入通知で購入された商品の情報を見ることができる。
